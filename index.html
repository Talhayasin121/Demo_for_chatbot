Zero-Cost Website Chatbot

A grounded company chatbot built like a small wiki.

This demo uses a retrieval-first approach inspired by the idea of turning company content into a searchable wiki. Instead of improvising answers, it responds from Orlando Auto Repair's website data and safely falls back when the site does not support a claim.

No paid API No LangChain Static deploy Strict guardrails

Why this design

Because you said you do not want to pay, this project does not use OpenAI, LangChain, Pinecone, or any hosted vector database. It is a static website chatbot that can be deployed free.

  • Answers are grounded in curated website content.
  • Unsupported claims are refused instead of hallucinated.
  • Common customer intents like hours, booking, services, and warranty are handled directly.
  • Everything can be hosted on GitHub Pages, Netlify, or Cloudflare Pages.
